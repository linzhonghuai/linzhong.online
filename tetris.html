Tetris

Classic block-stacking puzzle game

Game Rules

  • 🎮 Use arrow keys to move and rotate blocks
  • 🧱 Blocks fall from the top and can be moved and rotated
  • 💪 When a row is filled with blocks, it disappears and you score points
  • 🏆 The goal is to keep the game going as long as possible

Game Tips

  • ⚔️ Try to keep the bottom flat
  • 🛡️ Learn to use the "T-Spin" technique
  • 📈 Plan ahead for block placement
  • 🎯 Save I-shaped blocks for clearing four rows at once
